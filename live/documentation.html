Project documentation includes both a User Manual and Developer Web Site. The User Manual begins with an introduction to Watson that includes an overview of the labs and a brief history of their development. A detailed description of the features common to all labs is then presented, followed by separate chapters covering the functionality of the individual labs. The Developer Web Site links to the Subversion code repository and Mantis bug tracker.

Chapters in the Watson User Manual:

  • Chapter 01: Introduction to Watson
  • Chapter 02: General Watson Features
  • Chapter 03: Spreadsheet Lab
  • Chapter 04: Database Lab
  • Chapter 05: Data Structures Lab
  • Chapter 06: Graphics Lab
  • Chapter 07: Javascript Lab
  • Chapter 08: Object-Oriented Lab
  • Chapter 09: Assembly Lab
  • Chapter 10: Digital Logic Lab
